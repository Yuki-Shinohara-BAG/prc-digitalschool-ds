## 1. コニカミノルタとPPH事業

### コニカミノルタは何の会社？

- フィルム時代から続く光学・画像処理技術の会社
- 現在の主力は **オフィス向け複合機（コピー機・プリンタ）** と **プロダクション印刷機**
- 他にも医療用画像、計測機器、産業用印刷ヘッド等の事業がある

### PPH 事業とは？

- **P**rofessional **P**rint **H**eadquarters（プロフェッショナル印刷事業本部）
- 一般オフィスのコピー機ではなく、**業務として印刷を行う顧客** 向けの高速・高品質な印刷機を扱う
- 代表的なブランド：**AccurioPress**（アキュリオプレス）、**AccurioLabel**（ラベル印刷）

### 一言で

> オフィスのコピー機より「速く」「綺麗で」「大量に」「製本までできる」業務用印刷機を作って世界に売っている事業

公式：<https://www.konicaminolta.jp/business/products/graphic/index.html>

## 2. プロダクション印刷とは

### オフィス印刷との違い

| 観点 | オフィスコピー機 | プロダクション印刷機 |
| --- | --- | --- |
| 用途 | 社内資料・コピー | 商品としての印刷物 |
| 印刷量 | 月数千〜数万枚 | 月数万〜数百万枚 |
| 速度 | 30〜50枚/分 | 60〜140枚/分（ハイエンドは140枚クラス） |
| 用紙 | コピー用紙 | 厚紙・特殊紙・コート紙等 |
| 付加機能 | ステープル程度 | 中綴じ製本、無線綴じ、パンチ、断裁等 |
| 顧客 | 一般企業 | 印刷会社・出版社・企業の販促部門等 |

### プロダクション印刷の市場

- **デジタル印刷シフト**：従来は版を作るオフセット印刷だったが、デジタル印刷（版不要、即印刷）への置換が進む
- **小ロット・短納期**：「100部だけ・明日まで」のような注文に対応できる
- **可変印刷**：1枚ごとに宛名や内容を変えられる（DM等）

### 主要顧客の例

- 商業印刷会社（チラシ・パンフレット・冊子の請負）
- インハウス印刷（大企業の社内印刷部門）
- ラベル印刷会社（食品・化粧品等のラベル）
- 写真集・フォトブックの専門業者

## 3. PPHの売上構造（4本柱）

PPHの売上は、ざっくり4つの収益源から構成されている。

### ① ハード売り

- **本体**：印刷機そのもの。1台数百万〜数千万円
- **オプション**：給紙ユニット・フィニッシャー・大容量トレイ等。本体価格の数十%
- 売り方：
  - **新規顧客** への販売
  - **既存顧客** への入れ替え（リプレイス）販売
  - **オプション追加** 販売

### ② ソフト売り

- **Flux**：印刷ジョブのワークフロー管理ソフト
- **jobmanager**：印刷ジョブの一元管理
- 売り方：ハード商談時に「抱き合わせ」or 後追いで提案

### ③ 消耗品売り

- **トナー**：印刷の度に消費される。継続収益
- **用紙**：純正用紙を販売
- 売り方：サブスクリプション契約・定期便等

### ④ クリックチャージ

- **印刷1枚ごとにN円が販社に入る** モデル
- 顧客はハードを安く導入する代わりに、印刷量に応じて支払う
- 単価は機種・契約により異なる（**研修では仮値 @3円/枚** で計算）
- 販社経由でKMにも還元される **継続的な安定収益源**

### 売上構造の図解

```
┌─────────────── PPH 売上 ───────────────┐
│                                       │
│  ① ハード売り （単発の大型収益）       │
│   - 本体・オプション                    │
│                                       │
│  ② ソフト売り （単発+ライセンス）      │
│   - Flux、jobmanager                  │
│                                       │
│  ③ 消耗品売り （継続・リカーリング）   │
│   - トナー、紙                         │
│                                       │
│  ④ クリックチャージ （継続・印刷量連動）│
│   - 1枚N円 × 印刷枚数                 │
│                                       │
└───────────────────────────────────────┘
```

### 売上UPの「打ち手」と4本柱の関係

| 打ち手 | 主に効く収益源 |
| --- | --- |
| 新規顧客に機械を売る | ハード、後にクリック・消耗品も |
| 既存顧客の機械を上位機に入れ替え | ハード、クリック単価UP |
| オプション追加販売 | ハード、業務効率UPで印刷量UP |
| 顧客の印刷量を増やす（稼働支援） | クリック、消耗品 |
| 純正消耗品の使用率UP | 消耗品 |
| ソフトの抱き合わせ | ソフト |

### ⚠ 重要な前提：印刷可能枚数には「機械的な上限」がある

クリックチャージ収入を考えるときに必ず押さえておきたいのが、**1台が物理的に印刷できる枚数には上限がある** ということ。
顧客の業務需要だけでなく、機械側のスペックも収益の天井を決めています。

#### 物理上限を決める2つの要素

**① マシンスペック（PPM）**
- PPM（1分あたり印刷枚数）×　稼働時間 = 1日あたり物理上限
- 例：100 PPM 機 × 8時間稼働 = 1日 48,000枚 = 月換算 約100万枚
- ハイエンド機（140 PPM）と エントリー機（50 PPM）では、同じ稼働時間でも上限が **3倍近く違う**

**② オプションの付帯状況**
- **給紙ユニット（PFU/LCT）**：給紙容量が少ないと、用紙補充のたびに止まる → 連続稼働時間が短くなる
- **排紙スタッカー**：排紙容量が少ないと、回収のたびに止まる
- **フィニッシャー（中綴じ・無線綴じ等）**：仕上げ工程がボトルネックになり、本体の PPM が出ても全体スループットが落ちる
- **Transporter（搬送ユニット）**：機種間の搬送がボトルネックになるケースも

つまり、**「同じ機種でもオプション構成によって、現実的に出せる印刷量の上限は変わる」** ということ。

#### 物理上限と実際の印刷量のギャップ

```
　　物理上限（スペック × オプション）
　　　　│
　　　　│ ← この差が「施策で埋められる余地」
　　　　│
　　現在の実際の印刷量
```

- ギャップが大きい：稼働支援や教育で印刷量を増やせる余地が大きい（クリック施策の機会）
- ギャップが小さい：物理上限に近い ＝ 上位機へのアップグレード提案の機会
- ギャップがマイナス（実測 > 想定上限）：オプション追加や運用の工夫で無理して回している可能性

→ ロジックツリーで「印刷量を増やす」KPI を考えるときは、**「物理上限まで何%使えているか」** という視点を持っておくと、施策の意味づけが変わります。

## 4. 顧客像（誰に売っているか）

PPHの顧客は大きく3タイプ。

### 商業印刷会社

- **業態**：チラシ・パンフレット・冊子等を受注して印刷する会社
- **特徴**：印刷量が多い、品質要求が高い、納期が厳しい
- **狙い機種**：プロダクション機（PRESS6136等）

### インハウス印刷部門

- **業態**：大企業の社内印刷部門（製造業の取説、保険会社の帳票等）
- **特徴**：印刷量は多いが、毎月のボリュームが読みやすい
- **狙い機種**：プロダクション機〜ハイミドル機

### ラベル印刷会社

- **業態**：食品・化粧品等のラベルを印刷
- **特徴**：用紙が特殊（粘着紙）、印刷量は機種スペックに依存
- **狙い機種**：AccurioLabel（ラベル専用機）

### 顧客の購入意思決定者

- **経営層**：価格・投資回収
- **印刷現場**：操作性・品質・スピード
- **情シス・購買**：保守・サプライ調達

→ 提案時には「**誰に・何を訴求するか**」を分けて考える必要がある

## 8. 営業・販社・サービスの座組

PPH のビジネスは、コニカミノルタ単体ではなく **販社（販売会社）** と組んで顧客に届けている。

### 役割分担

```
┌─ コニカミノルタ（KM） ─┐
│  - 製品開発             │
│  - 製造                 │
│  - 販社向け卸           │
│  - 技術サポート         │
└──────┬──────────────────┘
       │
       ▼
┌─ 販社 ─┐
│  - 営業             │
│  - 設置・初期セットアップ │
│  - 日常サービス     │
│  - 消耗品供給       │
└──────┬──────────────┘
       │
       ▼
   ┌─ 顧客 ─┐
   │  印刷会社等  │
   └──────────┘
```

### データはどこに溜まる？

- **本体メーター**：印刷枚数・走行距離が機械に蓄積される
- **販社経由で収集**：定期的にカウンタ情報が KM のデータベースへ
- **設置構成情報**：オプション搭載状況・機種コード等
- → これらが本研修で扱う「データ」

## 9. ロジックツリーを書く準備運動

ロジックツリーは「**ゴール（KGI）→中間目標（KPI）→問題→原因→解決策**」と段階的に分解する道具。

### KGI と KPI の違い

- **KGI**（Key Goal Indicator）：最終的なゴール。「売上を上げる」
- **KPI**（Key Performance Indicator）：KGI を達成するための中間指標。「ハードを売る」「クリックチャージを増やす」等

### MECEを意識する

- **M**utually **E**xclusive, **C**ollectively **E**xhaustive
- 漏れなく、ダブりなく分解する
- 例：PPH 売上 = ハード + ソフト + 消耗品 + クリック（4つで全部、ダブりなし）

### 問題と原因を区別する

| 区別 | 例 |
| --- | --- |
| **問題**（What） | オプション追加販売が伸びない |
| **原因**（Why） | 顧客が活用方法を知らない／営業が説明できない |
| **解決策**（How） | 顧客向け活用デモ会／販社向け勉強会 |

### KPI は「数値で測れる」もの

| ✗ ぼんやり | ✓ 数値で測れる |
| --- | --- |
| 営業の質を上げる | 1営業あたり月間商談数 |
| サービスを改善する | サービスコール応答時間 |
| 顧客満足度を上げる | NPS（推奨度スコア）／解約率 |

### 解決策は「誰に・何を・どう」まで

| ✗ 抽象的 | ✓ 具体的 |
| --- | --- |
| 営業を頑張る | 同機種で印刷量が多い顧客リストを月次で営業へ渡す |
| マーケを強化 | インドのプロダクション機担当者向けに3ヶ月でデモ機N台投入 |

## 13. よくある初学者の疑問

### Q1: 機種が多すぎて覚えられません

A. **全部覚える必要はありません**。「プロダクション機（速い）」「ミドル」「エントリー」の3区分と、ラベル機を別物として扱う、くらいで十分です。

### Q2: なぜクリックチャージが重要なんですか？

A. ハード売りは1回限りの売上ですが、クリックチャージは **印刷し続ける限り永続的に入ってくる安定収益** だからです。10年間稼働すれば、ハード本体価格を上回ることもあります。

### Q3: ソフト売りや消耗品売りについて知識がないのですが…

A. 研修のメインは **ハード売りとクリックチャージ** です。ソフト・消耗品は「他にもこんな収益源がある」程度の理解で大丈夫です。

### Q4: 業界用語の英略称（ADF/PFU等）を全部覚える必要ありますか？

A. ロジックツリーを書く段階では「機種にはいろんなオプションがついている」が分かれば十分。データを触る2日目AM以降で必要に応じてユーザーズガイドを参照してください。

### Q5: KGI/KPIの違いがまだピンときません

A. **「KGI＝目的地、KPI＝途中のチェックポイント」** とイメージするとよいです。
- KGI「東京から大阪まで行く」
- KPI「名古屋を経由する」「燃費が15km/L以上を保つ」「8時間以内に着く」
